# 06 - 人物时间线：Jacques Perrin

调研日期：2026-05-26

## 时间线

| 时间 | 事件 | 思维影响 |
|---|---|---|
| 1941-07-13 | 出生于巴黎，本名 Jacques Andre Simonet。父亲与 Comedie-Francaise/TNP 剧场相关，母亲 Marie Perrin 是演员。 | 从小把表演、剧场、幕后劳动视为日常环境。 |
| 约 1956 | 青少年时期登台，进入戏剧世界；后进入巴黎国立高等戏剧艺术学院。 | 形成“存在于角色中”而非外部解释角色的演员观。 |
| 1961 | Valerio Zurlini 选中他出演 `La Fille a la valise`，成为电影主角。 | 意大利电影经验让他感到“到家”，打开跨国电影视野。 |
| 1964-1967 | 与 Pierre Schoendoerffer、Costa-Gavras、Jacques Demy 合作；出演 `La 317e section`、`Les Demoiselles de Rochefort`。 | 学到片场可以是纪律、冒险、诗性合奏。 |
| 1967 | 凭 `Un homme a moitie` 等获得威尼斯电影节最佳男演员奖（Coupe Volpi）。 | 演员身份得到国际确认，但他后来仍不满足于只做“镜子”。 |
| 1969 | 27 岁成为 `Z` 的制片人。 | 形成“制片人是电影律师”的职业伦理。 |
| 1972-1976 | 参与/制作 `Etat de siege`、`Section speciale`、`La Victoire en chantant`、`Le Desert des Tartares`。 | 在政治、历史、作者电影中建立高风险制片能力。 |
| 1982 | `Les Quarantiemes Rugissants` 商业失败并带来多年债务。 | 高风险创作的代价进入他的职业记忆。 |
| 1988-1989 | 出演 `Cinema Paradiso`；通过 Gérard Vienne 与 `Le Peuple singe` 深入自然纪录片方向。 | 从人类记忆的电影转向非人类生命的电影。 |
| 1994-1997 | 生产 `Microcosmos`，1997 年获得 Cesar 最佳制片等奖项。 | 建立“微观自然 + 技术创新 + 大银幕沉浸”的路线。 |
| 1999-2001 | `Himalaya` 获奥斯卡提名；`Le Peuple migrateur` 获 Fondation Gan 支持并上映。 | 形成跨地域、长周期、自然史诗化的制作模型。 |
| 2003 | `Le Peuple migrateur / Winged Migration` 获奥斯卡最佳纪录长片提名。 | 证明诗性自然电影可以进入全球主流视野。 |
| 2004 | 制作并出演 `Les Choristes`。 | 继续在大众情感电影与制片系统中发挥组织力。 |
| 2009-2011 | `Oceans` 上映并获 2011 Cesar 最佳纪录片。 | 将自然电影扩大到海洋系统与全球环保议题。 |
| 2015-2016 | `Les Saisons` 上映，讲述欧洲 15,000 年自然史。 | 从空间迁徙转向深时间视角。 |
| 2016-12-07 | 当选 Academie des beaux-arts 院士。 | 其跨演员、制片、自然电影的综合贡献获得制度认可。 |
| 2019-02-06 | 在 Academie des beaux-arts 安装入席。 | Jean-Jacques Annaud 将其定位为自然主义纪录片大片的少数实践者。 |
| 2022-04-21 | 在巴黎去世，享年 80 岁。 | 公共叙事集中回望他作为演员、制片人、自然电影作者的三重身份。 |

## 近年动态

- 2016：`Les Saisons` 上映。
- 2016：当选 Academie des beaux-arts。
- 2019：正式安装入席 Academie des beaux-arts。
- 2022：去世；法国与国际媒体发布悼念文章。
